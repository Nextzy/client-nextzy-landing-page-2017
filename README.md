# client-nextzy-landing-page-2017
After git pull please **install** first. Then start **development server** or **create a production build**.
## Installation  
Using npm  
```
npm install
```
Using yarn  
```
yarn
```

## Running a development server
Using npm  
```
npm start
```
Using yarn  
```
yarn start
```

## Create a production build
Using npm  
```
npm run build
```
Using yarn  
```
yarn build
```
